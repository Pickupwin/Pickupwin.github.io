BZOJ 4424 -- Fairy

发表于 2019-05-29 | 更新于 2019-06-17 | 分类于 图论 ， 二分图 | 评论数：

摘要

给出一张无向图

删去一条边使之成为二分图

求那些边可行

题面

解

没有奇环，随意

否则必删奇环的交

而对于交中的树边，若它在某个偶环中，则不能删

若奇环只有一个，则它对应的非树边也可删（易知这条非树边必不在偶环中）

统计一条边所在奇环偶环数——打标记

Code
